## 1. 产品概述

面向集团品牌公关部的舆情风险地图系统，将全国各地涉及公司的负面舆情按区域可视化呈现，辅助公关团队快速识别风险热点、追踪处置进度、统筹区域应对。

- **目标用户**：集团品牌公关部经理、区域公关负责人、舆情分析师
- **核心价值**：一图掌握全国舆情态势，分级分类处置风险事件，复盘总结优化策略

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 公关经理 | 企业账号登录 | 查看全局地图、标记事件分类、指派责任人、查看复盘报告 |
| 区域负责人 | 企业账号登录 | 查看所辖区域事件、更新处置进度、提交处置说明 |

### 2.2 功能模块

1. **风险地图首页**：品牌筛选、时间范围选择、舆情类型过滤、省级热力图、城市风险卡片详情、事件分类标记
2. **待处理清单页**：多维排序（扩散速度/情绪倾向/本地媒体参与度）、事件列表、晨会确认标记
3. **复盘分析页**：本周高风险地区排名、高频问题词云、已降温事件汇总、区域介入建议

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 风险地图首页 | 顶部筛选栏 | 品牌下拉选择、日期范围选择器、舆情类型多选标签 |
| 风险地图首页 | 中国热力地图 | 按省份/城市风险等级着色，hover显示风险数值，点击进入城市详情 |
| 风险地图首页 | 右侧风险卡片 | 显示选中城市的主要话题、首发平台、转发峰值、典型原文、媒体层级 |
| 风险地图首页 | 事件处置面板 | 事件分类标记（误解投诉/质量争议/劳动纠纷/监管关注）、责任人选择、处置状态切换 |

| 待处理清单页 | 统计概览 | 待处理总数、高风险数、今日新增、已确认数 |

| 复盘分析页 | 策略建议 | 基于数据自动生成统一回应建议、区域团队介入建议 |

## 3. 核心流程

用户进入首页后，通过筛选条件定位目标舆情范围，地图实时渲染各区域风险强度；点击具体城市查看详情并标记分类、指派责任人；每日晨会通过待处理清单逐条确认处置优先级；周度复盘通过分析页判断是否需要总部统一回应或派驻区域团队。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝 (#0A2540) 作为主色，体现专业稳重；辅以警示红 (#E63946)、注意橙 (#F4A261)、安全绿 (#2A9D8F) 表示风险等级
- **背景色**：深色模式为主 (#0F172A)，配合数据可视化的发光效果，营造指挥中心氛围
- **字体**：标题使用 "Noto Sans SC" 加粗，正文使用 "PingFang SC" / "Microsoft YaHei"，数字使用等宽字体增强数据感
- **布局**：顶部导航 + 左侧地图 + 右侧详情面板的三栏式指挥中心布局
- **动效**：风险脉冲动画、数据渐变过渡、面板滑入滑出、地图悬浮微光效果

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 风险地图首页 | 顶部筛选栏 | 品牌下拉选择器、日期范围选择器、标签式舆情类型筛选、重置按钮 |
| 风险地图首页 | 热力地图 | 中国地图轮廓、省级区块按风险等级填色、风险数值tooltip、选中城市高亮发光 |
| 风险地图首页 | 右侧详情卡 | 城市名称标题、风险等级徽章、主要话题列表、数据指标网格、原文引用块、操作按钮组 |
| 待处理清单页 | 顶部概览 | 四个数据卡片（待处理/高风险/今日新增/已确认）、排序切换按钮组 |
| 待处理清单页 | 事件列表 | 表格式列表、风险等级色块、情绪标签、扩散进度条、操作按钮列 |
| 复盘分析页 | 高风险地区 | 横向柱状图、省份名称、风险分数、环比变化箭头 |
| 复盘分析页 | 问题词云 | 大小渐变的词汇云、颜色深浅表示出现频次 |
| 复盘分析页 | 已降温事件 | 时间线式列表、风险等级下降标识、降温时长标签 |
| 复盘分析页 | 策略建议 | 卡片式建议、优先级标签、建议详情展开 |

### 4.3 响应式

- 采用桌面端优先设计，最低支持 1280px 宽度
- 平板端折叠右侧面板为抽屉式
- 移动端简化地图为列表视图

### 4.4 数据可视化

- 地图使用 SVG 路径绘制中国省级行政区
- 热力颜色从浅蓝（低风险）→ 黄色 → 橙色 → 红色 → 深红（极高风险）渐变
- 风险脉冲动画提示高风险区域
- 数据指标采用数字跳动动画效果
